# Keurduino

A simple Arduino sketch for interacting with a Keurig Coffee Maker.

|  Pin |    Function   |
|:----:|:-------------:|
| `A0` |   Heater LED  |
| `A1` |   Power LED   |
| `A2` |   Water LED   |
|  `3` |  Power Button |
|  `4` |  Small Coffee |
|  `5` | Medium Coffee |
|  `6` |  Large Coffee |

Attach the Analog Ground Pin to one of the green ground cables in the Keurig.

Rest will be filled in later.
